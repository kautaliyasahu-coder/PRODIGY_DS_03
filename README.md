Project Overview:

In this project, I built a Decision Tree Classifier to predict whether a customer will purchase a bank product or service based on their demographic and behavioral data.

The dataset used is the Bank Marketing Dataset from the UCI Machine Learning Repository. It includes customer information such as age, job, marital status, education, balance, and loan details.

Technologies Used:

Python

Pandas

NumPy

Scikit-learn

Matplotlib

Steps Performed:

1.Loaded and explored the dataset

2.Preprocessed the data and converted categorical values

3.Split the dataset into training and testing sets

4.Built and trained a Decision Tree model

5.Evaluated the model using accuracy score

Learning Outcome:

Through this project, I learned how to implement a Decision Tree algorithm, preprocess data, and build a basic machine learning model for classification.
